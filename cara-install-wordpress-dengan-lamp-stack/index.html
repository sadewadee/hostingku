Install WordPress dengan LAMP Stack

Seperti yang kita tahu, WordPress nyaris menguasai pasar CMS di dunia dengan Control Panel ataupun non- Control Panel pada server. Tidak hanya dari dashboardnya yang gampang dimengerti oleh pendatang baru, fitur plugins serta themes yang banyak dan mudah disesuaikan dengan kebutuhan. WordPress dapat digunakan untuk E- commerce, Retail, Magazine, Portfolio atau custom sesuai dengan kebutuhan.

Persiapan

Pastikan sudah memiliki akses root pada vps, agar dapat mengeksekusi perintah.

  • Pastikan sudah terinstall LAMP (Linux, Apache, MySQL dan PHP).
  • Domain sudah dipointing ke server.

Melakukan Instalasi WordPress

Step 1 – Create Database & Grant Permissions

Login ke server sebagai root lalu masuk ke MySQL Database dengan menggunakan user root.

$ mysql -uroot -p

Buat database dengan perinta Create Database:

MariaDB>  CREATE DATABASE wordpresslamp CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci;

Buat user baru dengan perintah Create User Database dan berikan izin untuk mengakses database dengan Grant Permissions.

MariaDB> GRANT ALL ON wordpresslamp.* TO 'user'@'localhost' IDENTIFIED BY 'webwp123';

Gunakan perintah Flush Privileges untuk reload perubahan dan keluar dari MySQL dengan perintah EXIT.

MariaDB> FLUSH PRIVILEGES;

MariaDB> EXIT;

Step 2 – Setup WordPress

Masuk kedalam directory /tmp dengan menggunakan perintah cd untuk dijadikan sebagai tempat mendownload file WordPress sementara.

$ cd /tmp

Download versi terbaru WordPress menggunakan perintah wget.

$ wget https://wordpress.org/latest.tar.gz

Extract file yang sudah didownload menggunakan perintah tar -xzf dimana artinya :

  • x : digunaka untuk memberi tahu agar tar mengekstrak file.
  • v : “v” adalah singkatan untuk “verbose.” Opsi ini akan mencantumkan semua file satu per satu di arsip.
  • z : Opsi z sangat penting dan memberi tahu perintah tar untuk membuka kompresi file (gzip).
  • f : Opsi ini memberi tahu tar bahwa Anda akan memberinya nama file untuk dikerjakan.
$ tar -xvzf latest.tar.gz

Buat file konfigurasi WordPress dengan me-rename wp-config-sample.php menjadi wp-config.php.

$ cp /tmp/wordpress/wp-config-sample.php /tmp/wordpress/wp-config.php

Buat directory website yang ingin di install WordPressnya.

$ cd /var/www/html/
$ mkdir lamp.ngebutsite.com

Ubah kepemilikan directory domain tersebut dengena menggunakan chown.

$ chown -R apache:apache /var/www/html/lamp.ngebutsite.com/

Copy semua file nya ke direktori website menggunakan perintah cp.

$ sudo cp -a /tmp/wordpress/. /var/www/html/lamp.ngebutsite.com/

Jangan lupa untuk melakukan Generate secret salt key WordPress.

$ curl -s https://api.wordpress.org/secret-key/1.1/salt/

/** MySQL database username */

/** MySQL database password */

/** MySQL hostname */

define( 'DB_HOST', 'localhost' );

Save lalu exit.

Step 3 – Install dan setup SSL

Agar website lebih aman, mari kita gunakan SSL (Secure Site Layer). SSL ini menggunakan LetsEncrypt karena gratis dan kompatible dengan sebagian besar web browser seperti chrome, firefox, opera, safari dan IE.

Pertama Install git untuk keperluan clone.

$ yum install git -y

Masuk ke directory /opt.

$ cd /opt

Lalu jalankan perintah git clone.

$ git clone https://github.com/letsencrypt/letsencrypt

Stop terlebih dahulu services Apache agar auto-installer dari LetsEncrypt bisa melakukan binding pada port 80.

$ service httpd stop

Berikan izin untuk melakukan eksekusi dengan perintah chmod +x, maka selanjutnya jalankan auto-installer LetsEncrypt.

$ chmod +x ./letsencrypt/letsencrypt-aut
$ ./letsencrypt/letsencrypt-auto certonly --standalone

Langkah selanjutnya adalah memasukan email dan nama domain dari website.

Enter email address (used for urgent renewal and security notices) (Enter 'c' to

cancel): info@ngebutsite.com

Please enter in your domain name(s) (comma and/or space separated)  (Enter 'c'

to cancel): lamp.ngebutsite.com

Certificate dan Chain-nya akan disimpan pada:

/etc/letsencrypt/live/lamp.ngebutsite.com/fullchain.pem

Private Key disimpan pada:

/etc/letsencrypt/live/lamp.ngebutsite.com/privkey.pem

Step 4 – Konfigurasi Apache

Tambahkan konfigurasi pada virtual host dan SSL di Apache Virtual Host Conf:

$ nano /etc/httpd/conf.d/example.com.conf

<VirtualHost *:80>

  ServerName lamp.ngebutsite.com

  ServerAlias www.lamp.ngebutsite.com

  Redirect permanent / https://lamp.ngebutsite.com/

</VirtualHost>

<VirtualHost *:443>

  ServerName lamp.ngebutsite.com

  ServerAlias www.lamp.ngebutsite.com

  <If "%{HTTP_HOST} == 'www.lamp.ngebutsite.com'">

    Redirect permanent / https://lamp.ngebutsite.com/

  </If>

  DirectoryIndex index.html index.php

  DocumentRoot /var/www/html/lamp.ngebutsite.com

  ErrorLog /var/log/httpd/lamp.ngebutsite.com-error.log

  CustomLog /var/log/httpd/lamp.ngebutsite.com-access.log combined

  SSLEngine On

  SSLCertificateFile /etc/letsencrypt/live/lamp.ngebutsite.com/fullchain.pem

  SSLCertificateKeyFile /etc/letsencrypt/live/lamp.ngebutsite.com/privkey.pem

  <Directory /var/www/html/lamp.ngebutsite.com>

      Options FollowSymLinks

      AllowOverride All

      Require all granted

  </Directory>

</VirtualHost>

Save dan exit.

Lanjutkan Membaca  Menghitung Trafik Pada Hosting

Konfigurasi diatas akan memberitahukan Apache untuk melakukan redirect dari http ke https dan dari www ke non-www pada domain lamp.ngebutsite.com tersebut.

Kemudian jalankan kembali service Apache nya.

$ systemctl start httpd

Step 5 – Install Module PHP

Ada beberapa module extension yang diperluka oleh WordPress agar berjalan dengan baik.

$ yum install php-fpm php-mysql php-mysqlnd php-pdo php-gd php-ldap php-odbc php-pear php-xml php-xmlrpc php-mbstring php-soap curl curl-devel -y

Restart Apache untuk mengenali perubahan.

$ systemctl restart httpd

Step 6 – Setup Installation

Selah sukses melakukan instalasi seperti diatas, selanjutnya silahkan akses domain, maka selanjutnya akan otomatis di bawa kehalaman seperti dibawah, silahkan masukan data-data pada form tersebut sesuai kebutuhan.

 

Jika sudah klik Install WordPress dan cek pada homepage dengan cara mengaksesnya.
