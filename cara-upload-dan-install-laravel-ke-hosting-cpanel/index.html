Cara Upload dan Install Laravel ke Hosting cPanel

Buat kamu developer yang terbiasa dengan website pasti sudah tidak asing lagi dengan yang namanya Laravel. Laravel merupakan salah satu framework berbasis PHP yang populer saat ini. Ada banyak framework PHP didunia ini misalnya Codeigniter, Yii, Symfony, Phalcon dll. Laravel ini adalah salah satu favorit para developer karena memiliki banyak keunggulan. Pada kali ini kita akan membahasa bagaimana cara install laravel di hosting berbasis cPanel dan versi Laravel yang akan kita gunakan adalah laravel 5.6.

Persiapan Upload Ke Shared Hosting

Sebelum meng-upload aplikasi Laravel yang sebelumnya sudah kita buat, langkah pertama adalah melakukan compressing pada file yang dibutuhkan untuk memudahkan saat meng-upload file ke dalam hosting. Mari kita compres semua folder dan file kecuali folder public.

Selanjutnya masuk kedalam folder Public lalu compress semua fila dan folder yang ada didalamnya. Langkah terakhir kita export database yang nantinya akan kita import pada hosting.

Install Ke Hosting cPanel

Pada bagian ini kita akan meng-upload aplikasi laravel yang sudah kita compress tadi ke hosting cPanel yang sudah kita siapkan, dengan menggunakan file manager bawaan dari cPanel. langkah pertama yang kita lakukan adalah membuat sebuah folder baru dengan nama core pada folder public_html seperti dibawah ini.

Selanjutnya Upload file semua yang sudah dicompress kedalam folder public_html yang ada di hosting yang sudah kamu siapkan, pindahkan file zip yang didalamnya tidak ada folder public kedalam folder core yang sudah kamu buat sebelumnya, setelahnya lakukan extract pada folder tersebut. Langkah terakhir adalah meng-extract file zip public yang ada di dalam folder public_html.

Menambahkan Database

Kamu dapat membuat database melalui menu Databases Wizard yang ada di cPanel lalu selanjutnya kamu import file database yang sebelumnya sudah kamu extract melalui PHPMyAdmin yang ada di cPanel. Untuk detail cara membuat database kamu bisa membaca artikel berikut.

Setting Laravel

Setelah melakukan upload file dan database kedalam hosting sekarang saatnya kita untuk melakukan konfigurasi pada laravel agar website laravel dapat berjalan dengan sempurna. Pertama buka file .env yang ada didalam folder core lalu sesuaikan dengan database yang sudah kita buat sebelumnya.

DB_CONNECTION=mysql
DB_HOST=localhost
DB_PORT=3306
DB_DATABASE=database_name
DB_USERNAME=database_username
DB_PASSWORD=password

Selanjutnya buka dan ubah file index.php yang ada didalam folder public_html seperti yang ada dibawah ini
