Kirim Email dengan Sendgrid di WordPress

Pernah mengalami masalah saat mengirim email namun tidak sampai tujuan?, seperti email reset password, aktivasi akun, informasi pembayaran, atau email penting lainnya, berikut adalah penjelasan mengapa hal itu bisa terjadi.

Kebanyakan penyedia jasa hosting menonaktifkan PHPMail dengan alasan menghindari SPAM, untuk kenyamanan bersama dan reputasi IP server maka solusi itu yang digunakan. Namun jangan khawatir untuk masalah tersebut, kita masih bisa mengirim email menggunakan SMTP dan tentunya lebih aman karena sudah pasti akan masuk ke inbox email tanpa harus diblokir oleh anti spam. Disini kita akan menggunakan Sendgrid sebagai SMTP sender.

Pada plan gratis kita diberikan 40.000 email untuk satu bulan pertama dan 100 email perhari untuk bulan berikut, lebih dari cukup untuk sebuah bisnis kecil atau UKM. Jika volume pengiriman email cukup banyak, anda bisa membeli plan premium mereka.

Daftar Akun Sendgrid

Agar bisa menggunakan layanan Sengrid kita diwajibkan untuk mendaftar terlebih dahulu dan mendapatkan sebuah API Key. Jika belum punya akunnya anda bisa mendaftar di https://sendgrid.com/free/. sebagai contoh, saya akan mendaftar menggunakan plan gratis.

selanjutnya anda akan dibawa kehalaman dan diminta untuk mengisi formulir pendaftaran dan akun.

Setelah Anda membuat akun, klik tombol Start sebelah Integrate using our Web API or SMTP relay.

.

selanjutnya anda akan diminta untuk memilih metode pengatura, silahkan klik tombol choose pada kolom SMTP Relay.

selanjutnya, masukan nama API Key anda lalu klik tombol Create Key.

Dan, kata kunci anda akan muncul dibawahnya, pastikan anda membuat salinannya di notepad karena anda pasti akan memerlukannya nanti.

Konfigurasi Sendgrid dengan WP Mail SMTP by WPForms.

Sekarang setelah akun SendGrid sudah disiapkan, saatnya kita melakukan konfigurasi pada website. sebelum melakukan bagian ini, pastikan anda sudah menginstall plugin WP Mail SMTP by WPForms.

Setelah WP Mail SMTP diinstal, masuk kedalam pengaturan WP Mail SMTP by WPForms. mari kita bahas satu persatu pengaturan yang ada pada halaman ini.

Mail.

kolom ini berisi alamat email yang akan mengirimkan semua email yang ada di website. anda bisa mengisi dengan info@namadomainsaya.com

From Name

From name secara default akan menggunakan nama website anda, ada baiknya mengubahnya sehingga terlihat lebih profesional.

Mailer

pilih sendgrid sebagai pemroses email anda, anda bisa menggunakan opsi yang lain. namun pada tutorial ini kita akan menggunakan sendgrid.

langkah terakhir adalah melakukan konfigurasi agar sendgrid dan website kita bisa saling berkomunikasi, sekarang kebagian Sendgrid dan masukan API Key yang sebelumnya sudah disalin ke notepad.

Selesai!,selanjutnya anda bisa menyimpan pengaturan yang sudah dibuat dan melakukan test email untuk memastikan bawah settingan anda sudah benar.
