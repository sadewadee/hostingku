Membuat Database MySQL di cPanel

Semua website yang bersifat dinamis pasti memerlukan sebuah database untuk menyimpan data website-nya. Cara membuatnya pun cukup mudah dan dapat dilakukan melalui cPanel. Pada tutorial kali ini saya akan membahas bagaimana caranya untuk membuat database di cPanel. Sebelum itu pastikan kamu sudah mempunyai username dan password untuk masuk ke-cPanel.

Apa yang Dibutuhkan?

Seperti yang saya sebutkan diatas, pastikan anda sudah memiliki yang ada dibawah ini :

  • Akses ke cPanel.
  • Segelas Kopi

Langkah – langkah pembuatan database Mysql di cPanel

Langkah 1: Login ke cPanel dan Pilih Menu MySQL Databases

Silakan login ke cPanelmu dengan menggunakan akses yang sudah diberikan oleh penyedia hosting, jika belum memiliki silahkan beli hosting terlebih dahulu dan pastikan menggunakan cPanel. Setelah masuk kehalaman dashboard cPanel, klik menu MySQL Databases seperti yang terlihat pada gambar di bawah ini.

Langkah 2: Masukkan Nama Database

Masukan nama database pada kolom yang sudah disediakan (tuliskan nama database tanpa spasi). Selanjutnya, klik tombol Create Database seperti di bawah ini.

Jika proses pembuatan database berhasil kamu akan dibawa ke halaman seperti dibawah ini.

Selanjutnya silahkan klik Go Back, maka database-mu sudah berhasil dibuat.

Langkah 3: Buat Username dan Password

Setelah proses pembuatan database selesai mari kita lanjutkan untuk membuat username. Nantinya username dan password ini berguna untuk mengakses kedalam database.

Scroll ke bawah dan kamu akan menemukan bagian Add New User. Masukkan username yang diinginkan pada kolom yang tersedia tanpa spasi dan usahakan unik. Selanjutnya masukan password pada kolom password dan password (again), gunakan kombinasi huruf, angka dan spesial karakter serta pastikan strength meter-nya di atas 90.

Kamu juga bisa menggunakan Password Generator agar lebih mudah dalam membuat password. Jika kamu menggunakan langkah ini pastikan kamu sudah meng-copy password-mu dan menyimpannya.

Klik pada tombol Use Password dan selanjutnya kamu akan diarahkan kembali form Add New User. Setelah itu klik tombol Create User. Selanjutnya kamu akan di bawa ke halaman seperti di bawah ini, lalu pilih Go Back.

Langkah 4: Tambahkan User ke Database

Setelah membuat Database dan User langkah selanjutnya adalah mehubungkan keduanya. Pada bagian Add User To Database, Pilih User dan Database yang sudah kita buat sebelumnya dengan meng-klik dropdown yang ada.

Setelah menekan tombol Add, akan tampil halaman privileges seperti yang terlihat pada Langkah 5.

Langkah 5: Memilih Privileges

Beri tanda centang pada bagian ALL PRIVILEGES, kemudian klik tombol Make Changes seperti gambar di bawah ini.

Selesai Database sudah siap digunakan.

Kesimpulan

Selesai!. Jika semua langkah sudah dilewati dengan baik, artinya kamu sudah berhasil membuat database, user, password serta berhasil menghubungkan database dan user yang ada. Sekarang database sudah siap digunakan untuk keperluan websitemu. Databas yang kamu buat saat ini masih kosong, kamu bisa mengisinya dengan membuat tabel dan kolom menggunakan mysql atau kamu bisa meng-import dari database mu yang sudah ada sebelumnya. Terima Kasih.
