Mengaktifkan TLS 1.1 dan 1.2 di Outlook Windows 7

Pada komputer Windows 7 dan Windows 8.0, aplikasi dibuat menggunakan WinHTTP (Windows HTTP Services) seperti Outlook, Word, dll. Hanya akan mendukung TLS 1.0. Akibatnya, jika kamu mencoba menggunakan koneksi ssl dari Outlook ke hostingmu, Outlook akan mengeluarkan pesan error” your server does not support the connection encryption type you have specified“.

Untuk menyelesaikan masalah, kamu bisa mengikuti langkah-langkah dibawah ini:

  1. Instal Update Windows KB3140245 , bisa melalui Windows Update di mana akan tersedia sebagai Optional Update, atau download dari Microsoft Update Catalog http://www.catalog.update.microsoft.com/search.aspx?q=kb3140245.
  2. Download file MicrosoftEasyFix51044.msi dari halaman berikut dan instal di komputermu https://support.microsoft.com/en-us/help/3140245/update-to-enable-tls-1-1-and-tls-1-2-as-a-default-secure-protocols-in

File ini tersedia untuk di-download pada bagian yang Easy fix pada halaman yang disebutkan di atas. Jika opsi diatas tidak bisa, kamu dapat menginstall registry yang sudah saya sediakan ke komputermu, silahkan download file nya disini
